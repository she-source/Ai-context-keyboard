# AI Content Keyboard

An AI-power-ued keyboard designed to understand conversational context and user tone, providing seamless, personalized reply suggestions in real time. Built as a portfolio project and submission for the LUMA Hackathon.

---

## 🚀 Key Features

* **Context-Aware Analysis:** Understands the mood, intent, and flow of the conversation before suggesting replies.
* **Smart Pop-up Suggestions:** Displays quick reply options in a clean, unobtrusive pop-up directly on your screen.
* **Personalized Tone Matching:** Learns and adapts to *your* unique writing style so suggestions actually sound like you—preventing over-explaining or overreacting.
* **Zero-Friction Workflow:** Eliminates the need to copy text, switch to external apps like ChatGPT, and paste it back.

---

## 💡 The Problem & Business Validation

### The Problem
Online communication is full of friction. Users frequently struggle with tone analysis, leading to over-explaining, overreacting, or social anxiety during real-time texting. Existing AI tools require users to copy text, leave their messaging app, open an AI tool, paste the text, generate a reply, copy it, and switch back. This breaks conversation flow.

### The Solution (MVP Market Angle)
The **AI Content Keyboard** embeds advanced LLM capabilities directly into the user's workflow. 
* **Target Audience:** Professionals, non-native speakers, and students navigating high-stakes or daily digital communication.
* **Monetization Strategy:** Freemium model—basic context replies are free; advanced personalized style-matching and multi-language support require a premium subscription.

---

## 🛠️ Tech Stack

* **Frontend Prototype:** Streamlit / Python (Simulating a mobile screen/keyboard interface)
* **AI Core Engine:** Advanced Natural Language Processing via Google Gemini API
* **Data Processing:** Python (Pandas/JSON) for managing mock conversation histories and localized context tracking
